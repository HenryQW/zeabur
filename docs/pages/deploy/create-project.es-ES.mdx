# Crear Proyecto

Para desplegar servicios en Zeabur, primero necesitas crear un proyecto.

Puedes seguir los pasos a continuación para crear un proyecto.

## ¿Qué es un proyecto?

Un proyecto es una colección de servicios que se despliegan juntos. Los servicios en el mismo proyecto pueden comunicarse entre sí mediante una red interna.
Puedes crear múltiples proyectos en diferentes regiones y desplegar varios tipos de servicios en cada proyecto.

Además, puedes [establecer el presupuesto y gestionar los usos](../manage/budget) del proyecto, [invitar a otros miembros del equipo](../manage/invite) para gestionar los servicios, y más.

## Cómo crear un proyecto

Primero, necesitas navegar a la [página de proyectos del panel de control](https://dash.zeabur.com/projects).

Si eres nuevo en Zeabur y no tienes ningún proyecto, verás una página de bienvenida para crear tu primer proyecto.

![Crear proyecto de bienvenida](/deploy/create-project/onboarding.png)

Si ya has creado proyectos antes, simplemente puedes hacer clic en el botón **Crear Proyecto** o presionar `CMD + K` o `Ctrl + K` para abrir la paleta de comandos y seleccionar **Crear Proyecto**, 
luego selecciona la región donde deseas crear el proyecto. El proyecto se creará automáticamente con un nombre aleatorio.

![Seleccionar Región](/deploy/create-project/select-region.png)

## ¿Qué sigue?

Verás un proyecto vacío después de seleccionar la región:

![Proyecto Vacío](/deploy/create-project/empty-project.png)

Puedes hacer clic en el botón **Desplegar Nuevo Servicio** para desplegar un servicio en este proyecto, o hacer clic en el botón de configuración para gestionar la configuración del proyecto e invitar a miembros del equipo.
