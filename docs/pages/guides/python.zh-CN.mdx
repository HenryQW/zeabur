# 部署 Python 项目

Zeabur 支持多种类型的 Python 项目，包括但不限于以下：

- [Flask](./python/flask)
- [Django](./python/django)
- FastAPI

如果您的源代码中有 `app.py`、`main.py` 等入口文件，Zeabur 也将自动启动。

## Serverless

Zeabur 支持将 Python 项目部署为 serverless 项目。但是目前需要 opt-in。
请参考 [启用 Serverless](/deploy/serverless) 页面启用 serverless 支持。
