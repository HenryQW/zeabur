# Pythonプロジェクトをデプロイ

Zeaburはさまざまな種類のPythonプロジェクトをサポートしており、以下に限らず、次のようなプロジェクトが含まれます：

- [Flask](./python/flask)
- [Django](./python/django)
- FastAPI

ソースコードに `app.py`、`main.py` などのエントリーポイントが含まれている場合、Zeaburは自動的に起動します。

## サーバーレス

ZeaburはPythonプロジェクトをサーバーレスプロジェクトとしてデプロイするサポートを提供しています。ただし、現在はオプトインが必要です。
サーバーレスサポートを有効にするには、[サーバーレスを有効にする](/deploy/serverless) ページを参照してください。
